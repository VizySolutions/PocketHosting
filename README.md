# PocketHosting

**PocketHosting** é um aplicativo Android desenvolvido pela **Vizy Solutions** para criar, gerenciar e executar servidores **PocketMine-MP** diretamente no celular.

O objetivo do app é permitir que desenvolvedores, donos de servidores e usuários avançados possam testar plugins, mundos e configurações do PocketMine-MP sem precisar contratar uma hospedagem externa ou usar um computador ligado o tempo todo.

---

## Sobre o aplicativo

O PocketHosting transforma o celular Android em um ambiente de testes para servidores Minecraft Bedrock usando PocketMine-MP.

Com ele, o usuário pode baixar, iniciar, parar, reiniciar e administrar um servidor PocketMine-MP diretamente pelo próprio aparelho, utilizando uma interface simples e prática.

O aplicativo foi pensado principalmente para:

- Desenvolvedores de plugins PocketMine-MP;
- Donos de servidores Minecraft Bedrock;
- Usuários que querem testar configurações localmente;
- Pessoas que não possuem hospedagem paga;
- Projetos de estudo, desenvolvimento e testes.

---

## Principais funções

- Criar e executar servidor PocketMine-MP no Android;
- Baixar a versão mais recente do PocketMine-MP;
- Iniciar, parar e reiniciar o servidor;
- Console em tempo real;
- Campo para enviar comandos diretamente ao servidor;
- Exibição de status do servidor;
- Exibição do IP local e porta;
- Exibição de jogadores online;
- Exibição da versão do PocketMine-MP;
- Gerenciador de arquivos integrado;
- Edição de arquivos do servidor;
- Envio de arquivos para o servidor;
- Sistema de idiomas;

---

## Para que serve?

O PocketHosting serve para facilitar a criação e o gerenciamento de servidores PocketMine-MP em dispositivos Android.

Ele pode ser usado para testar plugins, configurar mundos, editar arquivos do servidor, executar comandos pelo console e verificar o funcionamento do servidor antes de colocar o projeto em uma hospedagem profissional.

Também pode ser usado como uma solução simples para quem deseja rodar um servidor local na própria rede Wi-Fi.

## Requisitos

- Android 8.0 ou superior recomendado;
- Dispositivo Android 64-bit;
- Conexão com a internet para baixar os arquivos iniciais;
- Espaço livre no armazenamento interno;
- Minecraft Bedrock instalado em outro dispositivo ou no mesmo ambiente de rede para testes.

---

## Como usar

1. Instale o aplicativo no celular Android.
2. Abra o PocketHosting.
3. Execute a instalação/atualização dos arquivos necessários.
4. Toque em **Iniciar** para ligar o servidor.
5. Acesse a aba **Console** para acompanhar os logs em tempo real.
6. Use o campo de comando para executar comandos no servidor.
7. Acesse a aba **Arquivos** para editar configurações, plugins e mundos.
8. Conecte-se ao servidor usando o IP local exibido no aplicativo e a porta configurada.

---

## Observações

Este aplicativo foi criado para facilitar testes, estudos e desenvolvimento com PocketMine-MP em Android.

Para servidores públicos com muitos jogadores, uso comercial intenso ou ambientes de produção avançados, recomenda-se avaliar uma hospedagem dedicada, VPS ou servidor profissional.
